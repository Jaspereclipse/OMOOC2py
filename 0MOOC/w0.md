
#Use of Gitbook 


##问答区-Disqus

###1. 注册Disqus

登陆[Disqus](https://disqus.com/)，注册。

###2. 修改`book.json`

```
    {
        "plugins": ["disqus"],
        "pluginsConfig": {
            "disqus": {
                "shortName": "jasperspythonicjourney"
            }
        }  
    } 
```
